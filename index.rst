EScorpus es una aplicación para generar corpus de *textos* para problemas de NLP.

Overview
****************

Sunshine es una plataforma de gestión documental, orientada al concepto de los repositorios institucionales abiertos, que complementa además funcionalidades de los sistemas de recomendación. Utiliza las bibliotecas python-nltk para procesamiento de Lenguaje Natural, y python-whoosh como biblioteca de IR.

  Objetivo: Mejorar el rendimiento de la versión 0.x, utilizando menor tamaño de disco y conservando funcionalidades y velocidad.

Core & Arquitecture
*************************

Sunshine utiliza el concepto del modelo de las 3 capas, utilizando la arquitectura de Django: Modelo - Vista - Template. Tiene una arquitectura de configuración simple y está orientado a plugins. Sus tecnologías base son python y C++.

* :ref:`Dev_Doc`

Posee además un extenso conjunto de bibliotecas de python y códigos C++ que garantizan una comprensión más estandar de su código al utilizar bibliotecas populares y mantenidas por la comunidad internacional.

* :ref:`Libraries`
