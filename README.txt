Para entender este proyecto por favor lea:
~/html/index.html

Si usted ha obtenido el código fuente y quiere generar la documentación, ejecute el siguiente comando.
> sphinx-build -b html ./ html/

(debe tener instalado el paquete python-sphinx)
Esto re-generará la ayuda html dentro de la carpeta "html".

Toda la ayuda que usted necesita para instalar, usar este software, desarrollar algún componente nuevo, etc podrá encontrarlo en la ayuda.

Atte. Autores.
